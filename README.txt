Esta página fue desarrollada durante el curso "Programación Web desde 0 con PHP 8, Mysql, Laravel y Symfony"
impartido en la página de Udemy, con el objetivo de implementar los conocimientos de Html5 y Css3.

La página esta basada en una empresa de pasteleria, cuenta con 3 secciones inicio, sobre nosotros y contacto,
todas las secciones tienen sus archivos de estilos css asociados respectivamente.

-Inicio -> muestra informacion sobre los postres disponibles, y cuenta con enlaces para ponerse en contacto con
          la empresa para realizar un pedido y tambien conocer mas sobre la empresa.
-Sobre nosotros -> esta sección es la encargada de mostrar la información sobre la empresa.
-Contacto -> cuenta con un formulario para realizar un pedido.

Las imagenes utilizadas en esta página son de uso gratiuto, descargadas de pixabay.com y images.pexels.com, 
al igual que los iconos que fueron descargados de icon-icons.com.
